# 👕 Fashion MNIST Clothing Classifier

A Deep Learning project that classifies clothing images using a CNN model and provides predictions via a Streamlit web app.

## 🚀 Features
- CNN-based image classification
- Trained on Fashion MNIST dataset
- Streamlit UI for image upload
- Top-3 predictions with confidence scores

## 🧠 Tech Stack
- Python
- TensorFlow / Keras
- Streamlit
- NumPy
- Matplotlib
